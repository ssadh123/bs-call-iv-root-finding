# Implied Volatility Black-Scholes

This project computes implied volatility for at-the-money European call options under the Black-Scholes-Merton model with continuous dividends.

## What’s Included
- Newton’s method implementation
- Secant method implementation
- Bisection method implementation
- Approximate closed-form volatility estimate
- Iteration output and convergence comparison

## Notebook
- `HW05_Saumya_Sadh_CODE.ipynb`

## Goal
Given market option prices and model inputs (`S`, `K`, `T`, `r`, `q`), solve for the implied volatility `sigma` and compare numerical approaches.
